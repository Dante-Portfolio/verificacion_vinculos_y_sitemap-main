Verificación de vínculos

Googlebot navega de una URL a otra mediante la recuperación y el análisis de vínculos, mapas del sitio y redireccionamientos. Además, considera cada URL como la primera y única URL que ha visto de tu sitio. Para asegurarte de que Googlebot encuentre todas las URLs en tu sitio, haz lo siguiente:

Usa elementos <a> que Google pueda rastrear. Asegúrate de que sea posible acceder a todas las páginas del sitio con un vínculo en otra página que se pueda encontrar. Asegúrate de que el vínculo de referencia incluya texto o, en el caso de las imágenes, un atributo <alt> relevante para la página de destino.

Recomendado

<a href="https://example.com">
Google solo puede rastrear tu enlace si se trata de un elemento HTML <a> con un atributo href
<a href="/products.php?id=123">
Asegúrate de que la URL del elemento <a> se resuelva como una dirección web real
<a href="https://example.com">texto del enlace </a>
Coloca el texto del enlace entre elementos <a> para que Google pueda rastrearlo
<a href="https://example.com">Biblitografía completa</a>
El texto del enlace debe ser descriptivo y relevante
<a href="https://example.com">El quijote</a>
El texto del mensaje debe ser conciso
<a href="https://example.com/page1">seccion 1</a> texto entre enlaces ...
<a href="https://example.com/page1">seccion 2</a> texto entre enlaces ...
<a href="https://example.com/page1">seccion 3</a> texto entre enlaces ...
Enlaces separados por texto y en contexto

No recomendado

<span href="https://example.com">
Si no hay etiqueta <a> no podrá rastrearlo, aunque exista el atributo href en cualquier otro lemnto HTML
<a href="javascript:goTo('products')">
Si la URL no es una dirección como tal, no será indexada
<a href="https://example.com"></a>
Google puede usar el atributo title como texto del enlace si el elemento <a> está vacío
<a href="https://example.com">Más información</a>
El texto del enlace es totalmente generico
<a href="https://example.com"> En un lugar de la Mancha, de cuyo nombre no quiero acordarme, no ha mucho tiempo que vivía un ...</a>
El texto del enlace es excesivamente largo
<a href="https://example.com/page1">1</a> <a href="https://example.com/page1">2</a> <a href="https://example.com/page1">3</a> <a href="https://example.com/page1">4</a> <a href="https://example.com/page1">5</a>
No hay contexto y es totalmente genérico
